Definition 1.1.1. \(n\)-Dimensional Real Euclidean Space.

Define \(n\)-dimensional real Euclidean space to be the set of all lists of exactly \(n\) real numbers. In set builder notation,
\begin{equation*} \mathbb{R}^n=\left\lbrace \begin{bmatrix} a_1 \\ a_2 \\ \vdots \\ a_n\end{bmatrix}:a_i\in\bbr\right\rbrace, \end{equation*}
where \(n\in\bbn\) or \(n\) is a natural number.
